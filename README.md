## 🚀 Featured Project — Kris Hinova

> **AI-powered plant disease detection platform for farmers**

Kris Hinova lets farmers upload a photo of a diseased crop and get instant diagnosis, treatment recommendations, and agricultural guidance — in natural language, with voice support.

**Live:** [kris-hinova.vercel.app](https://kris-hinova.vercel.app)

| Layer | Tech |
|-------|------|
| Frontend | Next.js, Tailwind CSS |
| Backend | FastAPI, Python |
| AI Pipeline | LangChain, LangGraph, Groq |
| Voice | Deepgram STT/TTS |
| Infra | Docker, Vercel |

**Key features:**
- 🌿 Multi-turn RAG pipeline for agricultural Q&A
- 🎙️ Voice assistant for low-literacy farmers
- 🔄 LangGraph stateful agent for complex multi-step reasoning
- 🐳 Fully containerised with Docker
